Installation
------------

1. Just install the extension and be sure CSS Styled Content is installed
2. Goto your root page and "Create template for a new site", remove the default setup! (page = PAGE...)
3. set the includes (Info/Modify): 1st CSS Styled Content, 2nd FLUID Page Template
4. Select the "Backend Layout" (in your page settings => Appearance)
5. Start working in fileadmin/page
