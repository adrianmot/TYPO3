Extension Manual
================

Easytemplate for TYPO3 CMS

Create template files, styles, javascripts...
Configure your TYPO3 CMS via Typoscript

-----------------
What this is NOT:
-----------------

* A template engine
* A template containing grids
* Predefined Templates
